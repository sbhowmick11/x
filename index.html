Problem Statement:
You are given a string 'S'. Your task is to check whether the string is palindrome or not. Note: String 'S' is NOT case sensitive

{
  int i;
  for (i = 0; s[i]; i++) {
  if (s[i] >= 'A' && s[i] <= 'Z') {
  s[i] += 'a' - 'A';
    }
  }

bool checkPalindrome(char *s)
{
  toLowerCase(s);
  int i = 0;
  int j = strlen(s) - 1;
  while (i < j) 
  {
    if (s[i] != s[j]) 
    {
        return false;
    }
    i++;
    j--;
  }
  return true;
}

Explanation:
We are explaining the c++ code,the logics are similar:
1.Converting String to Lowercase:
The "toLowerCase" function takes a string as input and converts it to lowercase. It iterates over each character of the string and checks if it is an uppercase letter. If it is, it converts it to lowercase by adding the difference between the ASCII values of lowercase 'a' and uppercase 'A' to the character. This ensures that all uppercase letters are converted to their lowercase equivalents.
              
2.Alternative Approach:
An alternative approach mentioned is using the built-in function "tolower()" to convert the entire input string to lowercase. This function is available in many programming languages and simplifies the process by handling the conversion internally.
              
3.Checking Palindrome:              
The "checkPalindrome" function takes a string as input and checks whether it is a palindrome. It first converts the string to lowercase using the "toLowerCase" function mentioned earlier. It then initializes two pointers, "i" and "j," pointing to the beginning and end of the string, respectively.
              
4.Palindrome Checking Loop:
The function enters a while loop that continues until the two pointers meet or cross each other. In each iteration, it compares the characters at positions "i" and "j" of the string. If they are not equal, it means the string is not a palindrome, and the function returns false. The pointers are then moved towards the center of the string by incrementing "i" and decrementing "j."
              
5.Return True for Palindrome:
If the loop completes without finding any mismatched characters, it means the string is a palindrome, and the function returns true.
