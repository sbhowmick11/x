Problem Statement:
You are given a unsorted array. You must sort the array using heap sort algorithm with a time complexity of O(nlogn).

 C:

#include <stdio.h>
//To heapify a subtree rooted with 
//node i which is an index in arr[].
//N is size of the heap
void heapify(int arr[], int N, int i)
{
    //Initialize largest as root
    int largest = i;
    //left = 2*i +1
    int l = 2 * i + 1;
    //right = 2*i+2
    int r = 2 * i + 2;
    //If left child is larger than root
    if (l < N && arr[l] > arr[largest])
        largest = l;
    //If right child is larger 
    //than largest so far
    if (r < N && arr[r] > arr[largest])
        largest = r;
    //If largest is not root
    if (largest != i)
    {
        int temp = arr[i];
        arr[i] = arr[largest];
        arr[largest] = temp;
        //Recursively heapify the 
        //affected sub-tree
        heapify(arr, N, largest);
    }
}
//Main function to do heap sort
void heapSort(int arr[], int N)
{
    //Build heap (rearrange array)
    for (int i = N / 2 - 1; i >= 0; i--)
        heapify(arr, N, i);
    //One by one extract an element 
    //from heap
    for (int i = N - 1; i > 0; i--)
    {
        //Move current root to end
        int temp = arr[0];
        arr[0] = arr[i];
        arr[i] = temp;
        //call max heapify on the 
        //reduced heap
        heapify(arr, i, 0);
    }
}
//A utility function to print array of size n
void printArray(int arr[], int N)
{
    for (int i = 0; i < N; ++i)
        printf("%d ", arr[i]);
    printf("\n");
}
//Driver's code
int main()
{
    int arr[] = { 12, 11, 13, 5, 6, 7 };
    int N = sizeof(arr) / sizeof(arr[0]);
    // Function call
    heapSort(arr, N);
    printf("Sorted array is:\n");
    printArray(arr, N);
}

#include <iostream>
using namespace std;

void heapify(int arr[], int N, int i)
{
    //Initialize largest as root
    int largest = i;
    //left = 2*i +1
    int l = 2 * i+1;
    //right = 2*i+2
    int r = 2 * i + 2;
    
    //If left child is larger than root
    if (l < N && arr[l] > arr[largest])
        largest = l;
    
    //If right child is larger than largest
    //so far
    if (r < N && arr[r] > arr[largest])
        largest = r;
    
    //If largest is not root
    if (largest != i) {
        swap(arr[i], arr[largest]);
    
    //Recursively heapify the affected
    //sub-tree
    heapify(arr, N, largest);
    }
}
//Main function to do heap sort
void heapSort(int arr[], int N)
{    
    // Build heap (rearrange array)
    for (int i = N / 2 - 1; i >= 0; i--)
    heapify(arr, N, i);
    
    //One by one extract element from heap
    for (int i = N - 1; i > 0; i--) 
    {
        //Move current root to end
        swap(arr[0], arr[i]);
        
        //call max heapify on the reduced heap
        
        heapify(arr, i, 0);
    }
}

Explanation:
We are explaining the C++ code,the logics are similar:
1. The code demonstrates the implementation of the Heap Sort algorithm, which is used to sort an array in ascending order.

2. The `heapify` function is used to maintain the heap property of a subtree rooted at index `i` within the array `arr` of size `N`. It compares the element at index `i` with its left and right children, and swaps elements to ensure the largest element is at the root.
                
3. The `heapSort` function builds the initial heap by calling `heapify` for each non-leaf node in a bottom-up manner. It starts from the last non-leaf node and continues up to the root.
                
4. After building the heap, the function performs the sorting process. It repeatedly swaps the root element (largest value) with the last unsorted element, reduces the size of the heap by one, and calls `heapify` on the reduced heap to restore the heap property.
                
5. The `printArray` function is a utility function that prints the elements of the array.
                
6. In the `main` function, an example array `arr` is defined, and the size `N` is calculated. The `heapSort` function is called to sort the array in-place. Finally, the sorted array is printed using the `printArray` function.

Overall, the code demonstrates the steps involved in Heap Sort, including heap construction and sorting, to achieve an ascending order arrangement of the elements in the array.
